Dans le cadre du cours "Algorithmique avancée et programmation C", nous devions concevoir un compresseur utilisant l'algorithme de Huffman. Cette méthode consiste en l'utilisation d'un code à longueur variable pour représenter un symbole du fichier source. Le code est déterminé à partir de la fréquence d'apparition des symboles de source, un code court étant associé aux symboles de source les plus fréquents. Ainsi, un symbole fréquent à un code plus court qu'un caractère n'apparaissant qu'un nombre limité de fois.
Pour concevoir cet algorithme, nous avons utilisé un arbre binaire dont les feuilles sont des éléments du fichier source, ainsi le code correspondant à chaque caractère est le chemin permettant d?atteindre ces éléments.

Notre groupe étant formé de 5 personnes (Assvin, Leonard, Salim, Thomas et Sylvie), nous avons dû mettre en pratique nos compétences et connaissances afin de mener à bien ce projet d'une durée de 10 semaines. Dans ce rapport, nous expliciterons donc les étapes principales du projet, en suivant un cycle en V, en allant de l'analyse jusqu'aux tests.
